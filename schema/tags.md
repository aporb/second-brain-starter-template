# Tags

The tag taxonomy for this vault. Use only these tags unless you suggest a new one to the human.

## Domain Tags

| Tag | Description |
|-----|-------------|
| `technology` | Software, hardware, systems engineering |
| `science` | Research, experiments, discoveries |
| `business` | Strategy, operations, finance |
| `policy` | Regulations, governance, law |
| `design` | UX, visual, architecture |
| `health` | Medicine, wellness, biology |
| `education` | Learning, pedagogy, training |
| `culture` | Arts, media, society |

## Process Tags

| Tag | Description |
|-----|-------------|
| `research` | Active investigation |
| `synthesis` | Combined from multiple sources |
| `decision` | Records a decision made |
| `question` | Open question, not yet resolved |
| `contradiction` | Conflicting information across sources |
| `reference` | Lookup material, not narrative |

## Status Tags

| Tag | Description |
|-----|-------------|
| `active` | Currently in progress |
| `archived` | No longer active, kept for reference |
| `stale` | Needs review — may be outdated |

## Format Tags

| Tag | Description |
|-----|-------------|
| `article` | External article or paper |
| `book` | Book or book chapter |
| `video` | Video content |
| `podcast` | Audio content |
| `conversation` | Notes from a discussion |
| `documentation` | Technical docs |

---

**Adding new tags:** If you need a tag that doesn't exist, use the closest match and suggest the new tag to the human. The human adds it to this file.
